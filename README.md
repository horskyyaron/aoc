# Advent of code

These are my solutions to the aoc problems.\
Each folder represents the i-th problem, in each folder there will be a folder
for each programming language solution.

e.g.

```
aoc
├── 2023
│   └── javascript 
│       ├── 1
│       │   ├── input.md
│       │   ├── main.test.ts
│       │   ├── main.ts
│       │   ├── mini-input.md
│       │   ├── package.json
│       │   ├── package-lock.json
│       │   └── tsconfig.json
│       └── 2
│           ├── input.md //input file (data)
│           ├── main.ts
│           ├── package.json
│           ├── package-lock.json
│           └── tsconfig.json
└── README.md
```
