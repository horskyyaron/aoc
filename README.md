# Advent of code

These are my solutions to the aoc problems.\
Each folder represents the i-th problem, in each folder there will be a folder
for each programming language solution.

e.g.

```
aoc
├── 1
│   ├── file(resource for the question)
│   ├── go
│   │   └── sol.go
│   └── python
│       └── sol.py
```
